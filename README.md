# HoneyHomes Real Estate

HoneyHomes Real Estate is a modern, responsive web application for browsing and exploring property listings. Built with **React** and **Vite**, it showcases properties with detailed descriptions, galleries, floorplans, and agent contact features.

---

## Features

- Responsive property listings with hero images and galleries
- Detailed property information: bedrooms, bathrooms, square footage, type, and tenure
- Floorplan and interactive gallery lightbox
- Agent contact section with phone/email buttons and tour request form
- Mobile-optimized "Read More" functionality for property descriptions
- Tabs for easy navigation: Details, Floorplan, Location
- Modern UI with hover effects, shadows, and rounded cards

---

## Tech Stack

- **Frontend:** React, Vite  
- **Styling:** CSS (custom variables, responsive design)  
- **Icons:** Lucide-react  
- **Routing:** React Router  
- **State Management:** React hooks (useState, useEffect)
